Built-in safeguard

It also cleans up after the platforms.

Instagram and TikTok exports sometimes embed emoji as broken image glyphs instead of real Unicode. The tool strips those automatically, and logs every time it actually changes something, so nothing is silently altered.

>_

Every cleanup is logged. Genuine emoji are left untouched. The fix is visible, not invisible, the same principle I'd want in any agent doing real work inside someone else's system.
